== Description ==

Allow for seamless payment using the Velvpay platform. This plugin integrates Velvpay as a payment method in your WooCommerce store.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to WooCommerce > Settings > Payments and enable Velvpay.
4. Configure the settings with your Velvpay account details.

== Frequently Asked Questions ==

= How do I obtain my Velvpay keys? =

You can obtain your keys from the Velvpay dashboard after creating an account.

= What is the webhook for? =

The webhook is used to receive payment status updates from Velvpay.
